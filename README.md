## 주요 기능

- **장학금 검색**: 다양한 조건에 맞는 장학금을 검색할 수 있습니다.
- **장학금 추천**: 사용자의 프로필을 기반으로 맞춤형 장학금을 추천합니다.
- **장학금 정보 상세보기**: 장학금에 대한 상세 정보를 제공합니다.
- **장학금 신청 관리**: 장학금 신청 상태를 추적하고 관리합니다.
